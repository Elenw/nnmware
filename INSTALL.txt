Thanks for using this django application.

NNMWARE CMF @ 2012-2014

Install django-current, unidecode, oauth2, xlwt, openid, django-grappelli, PIL and ReportLab(recommended)

cd /usr/src
git clone https://github.com/nnmware/nnmware.git
cd /usr/src/nnmware
./syncdb
./runserver:8080

locate you browser at http://localhost:8080/admin/ ,
login and enjoy.


you may disable django-grappelli in settings.py
